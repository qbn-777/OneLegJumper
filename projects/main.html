Single-Leg Jump Control for the G1 Humanoid Robot

Project Overview

This project implements single-leg jumping for the G1 humanoid robot in NVIDIA Isaac Gym. Using the ASAP codebase—built on HumanoidVerse and Human2Humanoid—the robot learns stable, dynamic jumps through custom reward design and policy optimization. The modular framework enables efficient simulation and transfer. This work also serves as a hands-on exploration of modern frameworks for imitation learning in humanoid control.

Methods

  • ✓ Proximal Policy Optimization (PPO) with customized reward shaping for jumping behavior.
  • ✓ Simulation conducted in Isaac Gym with physics-based dynamics.
  • ✓ Action-space design focused on torque control and jump height optimization.
  • ✓ Curriculum learning strategy: starting from small hops and scaling up jump difficulty.
  • ✓ Evaluation based on motion stability, jumping height, and landing control.
