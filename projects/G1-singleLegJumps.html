Single-Leg Jump Control for the G1 Humanoid Robot

Project Overview

This project implements single-leg jumping for the G1 humanoid robot using reinforcement learning (PPO) in Isaac Gym. The robot learns dynamic and stable jumping motions through customized reward structures and policy optimization. The results demonstrate effective learning and stable motion control in simulation.
